Git parancsok

  • git init: Inicializálja a Git rendszert, új git repositoryt hozz létre
  • git clone: Egy bizonyos git repository elérése
  • git commit -m: Milyen változatatások történtek az adott repositoryban és ezt egy rövid üzenettel együtt
  • git add .: Módosítások után, ezzel a paranccsal jelezzük, hogy készen áll a rögzítésre
  • git branch branch_neve: Egy bizonyos repository egyik ága
  • git checkout branch_neve: Az ágak közötti váltást teszi lehetővé
  • git status: Az aktuális ágról(branch) mutat információkat
  • git push: Egy adott repositoryban történt változtatások feltöltése
  • git pull: Egy adott repository jelenlegi állapotának leszedése

Visual Studio Code - Extensions(Kiterjesztések)

  1. C/C++ (Microsoft): A C és C++ nyelvek teljes körű támogatása: kódkiegészítés (IntelliSense), hibakeresés és fordítás.
  2. C# (Microsoft): A C# nyelv alaptámogatása. A C# Dev Kit pedig egy komplett fejlesztői eszközkészlet, amely teljes értékű projektek és alkalmazások létrehozását segíti.
  3. Code Runner: Legegyszerűbben futtathatod vele a kódot (C, C++, Java, JavaScript, Python stb.) egy kattintással, anélkül, hogy össze kellene állítanod a projektet.
  4. Docker (Microsoft): Leegyszerűsíti Docker tárolók, rendszerképek és alkalmazások létrehozását, kezelésését és telepítését a VSCode-ban.
  5. Electron Builder: Segédlet Electron alkalmazások egyszerű csomagolásához és telepítőkészítéséhez különböző platformokra.
  6. Live Server: Egy fejlesztési lokális szervert indít, amely automatikusan újratölti a weblapot, amikor módosítod a HTML/CSS/JS kódot.
